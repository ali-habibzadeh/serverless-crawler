> A serverless crawler with Lambda, Dynamodb and Kinesis Firehose

## Install

```sh
yarn install
```

```sh
yarn build
```

## Deploy

Add AWS key and secret to github secrets (/settings/secrets):

`AWS_ACCESS_KEY`

`AWS_SECRET`

```sh
yarn deploy
```
